Password Folder Quick Start Guide

Password Folder is designed to password-protect your folders and files from being seen, read or modified in Windows 7, Vista, XP and Server 2008, 2003. It works like a safety box, just drag and drop the folders or files you want to hide or protect into Password Folder, then no one can see, read or modify them.

Before you can use and unlock the functions of Password Folder, you have to set a password for Password Folder when you initially use it.

You can change the password for the program in Options. However, please remember the password for Password Folder as you need it to open and use the program. You can also set a password hint in case you may forget your password.

Please refer to below to use Password Folder.

  1. Click Options to set the Lock Options, you just need to click the green button to set the status and apply the settings.
    Note: This setting will apply for all files and folders you added into the program.

  2. You can easily drag and drop the files and folders you want to protect to the red rectangle area or click “Add” button to add into this program.
  3. Some system files and folders should not be protected as they will cause system error while the system can not access them. To avoid this, we add these files and folders to the "Exclude List". Moreover, you should also add other items which will cause such problems to the "Exclude List".
  4. You can directly click “Open in safe box” to open the protected files. And then you can read even edit the protected files.

    For protected folders, you can also click "Open in safe box" to open the File Browser, in which, you are enabled to view and edit the content.
